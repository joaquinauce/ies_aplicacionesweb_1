# Tienda Nova

**Nombre y apellido:** Completar con tus datos

## Descripción del proyecto

Este proyecto corresponde a la primera etapa de una aplicación web para una tienda online.
Incluye una página principal (`index.html`), una página de login, una página de registro y
tres páginas de categorías de productos: Electrónica, Indumentaria y Hogar.

## Estructura del proyecto

- `index.html`: página de inicio con navbar, icono de tienda, botón de logout y acceso a categorías.
- `login.html`: formulario de acceso con email y contraseña.
- `registro.html`: formulario de registro con nombre, apellido, email, contraseña y fecha de nacimiento.
- `categorias/`: páginas individuales para cada categoría.
- `styles.css`: estilos generales del sitio.

## Objetivo

Generar la estructura base de un ecommerce cumpliendo con los requisitos solicitados para la entrega.
